# Cheremsha
Командный проект "ML-сервис для предсказания ипотеки"


## Backend

### Общая структура
Веб сервис создан с помощью FastAPI и моделей Pydantic. Есть основные три пути:
* /upload-model для загрузки модели .pkl для предсказывания одобрения ипотеки
* /predict для предсказания по списку признаков (json)
* /predict-from-csv для предсказания по файлу данных (.csv)

Для каждого запроса использовалась встроенная модель FastAPI или собственная, написанная с помощью Pydantic.

### Тесты
Тесты для сервиса были написаны с помощью встроенного модуля FastAPI TestClient и библиотеки Pytest. Файл конфигурации: pytest.ini.

### Логирование
Логирование реализовано с помощью перенаправления логов uvicorn из терминала в файл с помощью. Файл конфигурации: log.ini.
